# GDP
Ghosts of Disasters Past

This is a web app to document natural disasters in select cities. Users can add, delete and edit disasters.

1. San Francisco
2. Los Angeles
3. Seattle
4. New York
5. Miami
6. Oklahoma City
7. Toronto

The natural disasters that can be documented are:
  *Earthquakes
  *Hurricanes
  *Tornadoes
  *Floods

## Technologies Used
*MongoDB
*Express
*Node.js
*CSS
*HTML
